### Emphasis

### Strong emphasis

### Code inline

Call the `function_name` function.

### Code blocks

Look at this output:

```
$ ls /usr/
bin        lib        libexec    local      sbin       share
```

Look at this code:

```python
def fib(n):
    if n <= 2:
        return 1
    else:
        return fib(n-1) + fib(n-2)
```
